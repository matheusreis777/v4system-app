## 3. Dados Coletados

### 3.1 Dados Fornecidos Diretamente pelo Usuário

Ao utilizar o V4 System, coletamos os seguintes dados pessoais fornecidos por você:

#### Autenticação

- CPF
- Senha (criptografada)

#### Perfil do Usuário

- Nome completo
- Telefone
- Empresa vinculada
- Perfil de acesso

#### Dados Operacionais

- Clientes (nome, email, CPF/CNPJ, telefone)
- Movimentações de vendas (dados do cliente, placa, status, observações)
- Veículos (placa, marca, modelo, ano, km, cor, combustível, preço)
- Leads (nome, telefone, CPF/CNPJ, observação)

#### Notificações

- Histórico de notificações lidas/não lidas
- Preferências de notificação

### 3.2 Dados Coletados Automaticamente

- **Informações do Dispositivo:** Modelo, sistema operacional, identificadores únicos
- **Dados de Acesso:** Logs de login, data/hora de acesso, duração da sessão
- **Localização:** Apenas se você conceder permissão (não coletado por padrão)
- **Analíticos:** Funcionalidades utilizadas, erros, performance

### 3.3 Dados de Terceiros

Podemos receber dados de empresas clientes que integram seus sistemas com o V4 System.

## 6. Compartilhamento de Dados

### 6.1 Quando Compartilhamos Dados

- **Empresa Contratante:** Seus dados são compartilhados com a empresa cujos serviços você utiliza
- **Processadores de Dados:** Provedores de hospedagem, serviços de segurança
- **Requisições Legais:** Quando obrigado por lei, ordem judicial ou autoridade pública

### 6.2 Quando NÃO Compartilhamos

- Nunca vendemos dados pessoais para terceiros
- Não compartilhamos com fins de marketing sem sua autorização
- Não realizamos compartilhamento para publicidade direcionada

### 6.3 Sigilo e Confidencialidade

Compartilhamos dados apenas com fornecedores que:

- Assinam Termos de Processamento de Dados
- Implementam medidas de segurança equivalentes
- Usam dados exclusivamente para fins contratados

---

## 7. Retenção de Dados

### 7.1 Período de Retenção

- **Dados de Autenticação:** Mantidos durante o uso ativo da conta + 90 dias após cancelamento
- **Dados Operacionais:** Re tidos conforme exigido pela empresa contratante e legislação
- **Logs de Acesso:** Mantidos por 12 meses para auditoria e segurança
- **Dados de Notificação:** Mantidos por 180 dias

### 7.2 Exclusão de Dados

Ao cancelar sua conta:

- Dados pessoais são anonimizados dentro de 30 dias
- Dados obrigatórios por lei são retidos conforme necessário
- Você pode solicitar exclusão acelerando o processo

---

## 8. Segurança dos Dados

### 8.1 Medidas Técnicas

- **Criptografia:** Dados sensíveis (senha, telefone) são criptografados em trânsito (HTTPS/TLS) e em repouso
- **Autenticação:** Tokens JWT com expiração configurada
- **Armazenamento Seguro:** expo-secure-store para dados críticos
- **Isolamento:** Dados separados por empresa e usuário

### 8.2 Medidas Administrativas

- Acesso restrito a dados pessoais (princípio de mínimo acesso)
- Treinamento de segurança para equipe
- Termos de confidencialidade com funcionários
- Política de senhas fortes

### 8.3 Incidentes de Segurança

Em caso de violação de segurança que comprometa dados pessoais:

- Notificaremos você dentro de 72 horas (conforme LGPD)
- Descreveremos a natureza da violação
- Indicaremos medidas de proteção tomadas

## 10. Cookies e Rastreamento

O V4 System utiliza:

- **AsyncStorage:** Para armazenar dados de sessão localmente no dispositivo

Podemos coletar dados analíticos para:
